NIX labs

Combining a hand-craftred Tassie Oak frame, laser-cut acrylic, and Soviet display tubes, NIXlabs designs and manufacturers these nixie clocks in Brisbane, Australia. Featuring a USB setup webapp, WiFi with Alexa/Google integration and a supercap time/date backup, they fuse retro display technology with a modern design.

LabVIEW Power Source with Test Harness Interface
(2018)

Can be used stand-alone or under LabVIEW control for automated testing, with 8 analog/digital IO and 4 digital-only IO for DUT monitoring. Command set includes adjustable slew rate, programmable over-current behaviour, and voltage flicker/interruption test sequences.

IR Remote Control Translator
(2020)

This device receives infra-red remote control signals and translates certain codes to other protocols in order to allow one remote to control multiple devices. A relay output with Li-ion backup additionally allows control over a mains power source.

Energy Harvesting Datalogger
(2015)

Featuring a large 128x64 matrix LCD and solar energy harvesting, this datalogger is USB connected and has 32MB of on-board flash memeory. Built-in sensors include temperature, humidity, barometic pressure, ambient light and ambient noise, with an expansion port exposing analog/digital GPIO, I2C and SPI.

DC Electronic Load
(2012)

A simple but extremely useful electronic load with modulation input allowing arbitrary load waveforms using an external source.

Medium-Votlage DC Source
(2017)

Essential for testing nixie tubes, this 50-300V DC source features a digital control loop with overload detection, and an RS232 control interface.

Simple Desktop Clock
(2007)

An early project, this colourful clock shows 24-hour time with automatic night-time dimming.

5-Channel Audio Power Amplifier with Control Unit
(2012)

This 5-channel 300W class AB audio amplifier and control unit features five input sources, digital tone control, sleep timer and remote control.
